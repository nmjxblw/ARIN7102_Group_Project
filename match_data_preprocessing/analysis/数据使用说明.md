## 二、合并增强表 `enhanced_drug_table.csv`

### 表概述

- **主键**: `drug_name`（药品名称，统一小写）
- **目的**: 以药品为中心，整合其属性、适应症、与 `disease_keys.json` 的疾病匹配，支持下游药物推荐
- **存储路径**: `match_data_preprocessing/data/enhanced_drug_table.csv`

### 字段说明

| 字段名 | 数据类型 | 来源表 | 描述 |
|--------|---------|--------|------|
| `drug_name` | str | DS1 + DS2 + DS4 | **主键**。药品名称，统一为小写 |
| `generic_name` | str | DS2 | 药物通用名/化学成分名（如 isotretinoin） |
| `drug_classes` | str | DS2 | 药物分类（如 Tetracyclines, Topical acne agents） |
| `brand_names` | str | DS2 | 品牌名列表（如 Accutane, Claravis） |
| `side_effects` | str | DS2 | 副作用描述文本 |
| `activity` | str | DS2 / DS3 | 药物活跃度百分比（如 87%） |
| `rx_otc` | str | DS2 / DS3 | 处方药(Rx) / 非处方药(OTC) 标识 |
| `pregnancy_category` | str | DS2 / DS3 | 妊娠安全分类（A/B/C/D/X/N） |
| `csa` | str | DS2 | 管制药物分级 |
| `alcohol` | str | DS2 | 酒精交互警告（X=不可同时使用） |
| `related_drugs` | str | DS2 | 相关/替代药物链接 |
| `avg_rating` | float | DS2 + DS4 | 综合加权平均评分（1-10分） |
| `total_reviews` | int | DS2 + DS4 | 评价总数 |
| `original_conditions` | str (JSON List) | DS1 + DS2 + DS4 | 原始适应症/疾病名称列表（未标准化的原始数据） |
| `medical_condition_description` | str | DS2 | 适应症的医学描述（来自 drugs.com） |
| `matched_disease_keys` | str (JSON List) | 模糊匹配生成 | 精确匹配到 `disease_keys.json` 中的疾病名列表；无匹配时为 `["others"]` |
| `matched_symptoms` | str (JSON List) | DS5 反推 | 通过 matched_disease_keys → DS5 反推出的症状关键词列表（如 `["itching", "skin_rash"]`） |
| `symptom_severity` | str (JSON Dict) | DS8 映射 | matched_symptoms 中每个症状的严重度权重（如 `{"itching": 1, "skin_rash": 3}`） |
| `disease_description` | str | DS6 | 匹配到的 disease_keys 疾病的标准描述文本 |

### 字段来源追溯

```
DS2 (drugs_side_effects)   ──→ drug_name, generic_name, drug_classes, brand_names,
                                side_effects, activity, rx_otc, pregnancy_category,
                                csa, alcohol, related_drugs,
                                medical_condition_description, rating, no_of_reviews

DS1 (drug-prescription)    ──→ 补充 drug_name ↔ condition 映射关系

DS4 (kuc-hackathon)        ──→ 补充 drug_name ↔ condition 映射 + 用户评分聚合

DS3 (drugs-common)         ──→ 补充 activity, rx_otc, pregnancy_category（DS2缺失时回填）

DS5 (disease→symptoms)     ──→ matched_symptoms（通过 matched_disease_keys 反推对应症状）

DS8 (symptom-severity)     ──→ symptom_severity（为 matched_symptoms 中的症状附加权重）

DS6 (symptom_Description)  ──→ disease_description（按 matched_disease_keys 关联）

disease_keys.json          ──→ matched_disease_keys（模糊匹配目标）
```
